# ulff-pl

This is a [Next.js](https://nextjs.org/) based personal web page.

---

## Running locally

Clone the GitHub repo, and run the commands:

```bash
npm i
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
